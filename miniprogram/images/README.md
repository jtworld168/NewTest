# 小程序图标说明

tabBar 图标和占位图需要替换为实际的 PNG 图片文件（建议 81x81px）：

## tabBar 图标
- `tab-home.png` / `tab-home-active.png` - 首页图标
- `tab-category.png` / `tab-category-active.png` - 分类图标
- `tab-cart.png` / `tab-cart-active.png` - 购物车图标
- `tab-order.png` / `tab-order-active.png` - 订单图标
- `tab-profile.png` / `tab-profile-active.png` - 我的图标

## 其他图片
- `default-avatar.png` - 默认用户头像
- `product-placeholder.png` - 商品默认图片

> 可以从 [iconfont.cn](https://www.iconfont.cn/) 下载图标
